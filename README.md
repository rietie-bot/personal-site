# 🐾 OpenClaw Setup Night — Event Page

Landing page for the [OpenClaw Setup Night](https://luma.com/1du8xbfg) in Hamburg.

**Stack:** Next.js 15 · React 19 · Tailwind CSS v4 · Motion-Primitives

## Quick Start

npm run dev
