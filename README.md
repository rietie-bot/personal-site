# 🌐 Personal Website Template

A clean, minimal personal website you can set up in minutes. Built for the **OpenClaw Setup Night** — but anyone can use it.

**Stack:** Next.js 15 · React 19 · Tailwind CSS v4 · Motion animations

## 🚀 Quick Start

### 1. Fork this repo

Click the **Fork** button at the top right of this page.

### 2. Clone your fork

```bash
git clone https://github.com/YOUR_USERNAME/personal-site.git
cd personal-site
```

### 3. Install & run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you should see the template running.

### 4. Make it yours

Edit these files (they're all in the `app/` folder):

| File | What to change |
|------|----------------|
| `app/data.ts` | Your projects, work experience, social links, email |
| `app/header.tsx` | Your name and title |
| `app/page.tsx` | Your bio/intro text |
| `app/layout.tsx` | Page title and meta description |

Look for the ✏️ emoji in comments — those mark the spots to edit.

### 5. Deploy

The easiest way: push to GitHub and enable **GitHub Pages** with the `gh-pages` branch, or deploy to **Vercel** for free:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Frietie-bot%2Fpersonal-site)

## ✨ Features

- **One-page layout** — clean, focused, no clutter
- **Dark mode** — automatic, with manual toggle
- **Smooth animations** — powered by Motion-Primitives
- **Fully responsive** — looks great on any device
- **Easy to customize** — just edit `data.ts` and you're done
